`useCheckboxGroup` is a custom hook used to provide checkbox group
functionality, aswell as state and focus management to custom checkbox group
components when using it.

## Import

```js
import { useCheckboxGroup } from '@chakra-ui/react'
```

## Return value

The `useCheckboxGroup` hook returns following props

| Name               | Type                            | Description                                     |
| ------------------ | ------------------------------- | ----------------------------------------------- |
| `value`            | `boolean`                       | A function to set the boolean value to `true`.  |
| `isDisabled`       | `boolean`                       | A function to set the boolean value to `false`. |
| `onChange`         | `(input: EventOrValue) => void` | A function to negate the boolean state.         |
| `setValue`         | `() => void`                    | A function to negate the boolean state.         |
| `getCheckboxProps` | `() => void`                    | A function to negate the boolean state.         |

## Usage

```jsx
function Example() {
  return <Box>Example goes here</Box>
}
```
