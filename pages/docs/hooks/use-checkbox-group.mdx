`useCheckboxGroup` is a custom hook used to provide checkbox functionality,
aswell as state and focus management to custom checkbox components when using
it.

## Import

```js
import { useCheckboxGroup } from '@chakra-ui/react'
```

## Return value

The `useCheckboxGroup` hook returns following props

| Name     | Type         | Description                                     |
| -------- | ------------ | ----------------------------------------------- |
| `on`     | `() => void` | A function to set the boolean value to `true`.  |
| `off`    | `() => void` | A function to set the boolean value to `false`. |
| `toggle` | `() => void` | A function to negate the boolean state.         |

## Usage

```jsx
function Example() {
  return <Box>Example goes here</Box>
}
```
